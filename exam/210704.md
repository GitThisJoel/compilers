### a) 

Lookup and decl.

```java
inh ClassDecl ClassDecl.lookup(String s);
syn ClassDecl Module.getChildren().lookup(String s) {
    for (ClassDecl cd : getClassDecls())
        if (cd.getID().equals(s))
            return cd;
    return null;
}

syn ClassDecl ClassDecl.decl() {
    return lookup(getID());
}
```
